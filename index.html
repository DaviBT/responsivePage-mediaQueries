Responsividade, pontos críticos

Não use grandes elementos de largura fixa, sempre utilize o viewport como ponto de referência, isso também se aplica a DIVs, imagens e botões.

Não deixe que o conteúdo dependa de uma determinada largura de visor para renderizar bem. Sempre considerar imagens que possuam um tamanho e qualidade mínima para o dispositivo de maior tela.

Usar medias queries para definir a larura dos elementos de acordo com o dispositivo, afim de impedir que os elementos ultrapassem o tamanho da tela.
